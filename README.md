# THUNA Public Portal

Next.js and Tailwind CSS implementation of the THUNA public-facing portal based on the stitched UI mockups and working PRD.

## Included routes

- `/` dashboard and analytics
- `/public-data` searchable case records
- `/public-data/[caseId]` citizen-facing case detail and timeline
- `/crime-map` interactive hotspot map view
- `/news` verified alerts and announcements
- `/login` OTP-style public access screen

## Run locally

```bash
npm install
npm run dev
```
