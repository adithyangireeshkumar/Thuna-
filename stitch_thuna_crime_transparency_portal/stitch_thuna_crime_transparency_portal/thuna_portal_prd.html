Design Style: Minimalism in Kerala Police Theme Product: THUNA — Crime Transparency Portal Design Philosophy: Minimalism + Government Authority + Public Trust Primary Goal: Fast, clear, trustworthy public information access 1. 🎯 Design Principles - Minimalism First: Essential elements only, no clutter, high readability. - Kerala Police Inspired: Khaki (#C3B091) and Deep Navy Blue (#0B2A4A) palette, official/professional look. - Accessibility Focus: High contrast, large text, mobile-friendly, keyboard nav. 2. 🎨 Color Palette - Primary: Kerala Police Khaki (#C3B091), Deep Navy Blue (#0B2A4A), White (#FFFFFF). - Secondary: Success Green (#2E7D32), Warning Orange (#F57C00), Danger Red (#C62828), Light Gray (#F4F6F8). 3. 🅰 Typography - Primary Font: Inter (Fallback: Roboto, Arial, Sans-serif). - Sizes: Heading 28px (Bold), Subheading 20px (Medium), Body 16px (Regular), Small 14px. 4. 🏠 Home Page UI - Header: Logo, Government label, Login button. - Hero: Search FIR/Case with station, type, and status filters. - Statistics: Cards showing Total, Today's, and Active cases. 5. 📄 Case List Page UI - Search/Filters + Case Table (FIR Number, Date, Crime Type, Station, Status, Action). 6. 📄 Case Details Page UI - FIR Information Card, Status Badge (Open/Orange/Green), Officer Info, Follow-up Timeline, News Articles, Download FIR PDF. 7. 🔐 Login Page UI - Centered minimal design, Google/Mobile OTP options.
